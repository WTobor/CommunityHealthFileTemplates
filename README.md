# CommunityHealthFileTemplates
GitHub templates for community health files:
- `CODE_OF_CONDUCT`
- `CONTRIBUTING`
- `ISSUE_TEMPLATE`
- `PULL_REQUEST_TEMPLATE`
- `README`
- `SUPPORT`

GitHub wil use and display this template files, when you add them to one of the locations:
- root folder
- the `.github` folder
- the _docs_ folder
